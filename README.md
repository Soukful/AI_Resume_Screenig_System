# AI_Resume_Screenig_System
NLP-based Resume Screening System that evaluates resumes against job descriptions and generates ATS scores with skill gap analysis.


Problem Statement :
Recruiters manually screen hundreds of resumes for a single job opening. This process is time-consuming and may overlook suitable candidates. The goal of this project is to automate resume screening by matching resumes with job descriptions using NLP and Machine Learning techniques.
